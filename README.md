# JBMM — Jai Bharat Mitra Mandal Collection App

Cross-platform (React Native + Expo) donation collection app for the annual
Ganesh Festival drive, with an offline-first volunteer collection flow,
auto-generated numbered PDF receipts, and an admin dashboard.

- `frontend/` — Expo app (see `frontend/README.md`)
- `backend/` — FastAPI + MongoDB API (see `MIGRATION_GUIDE.md` for setup
  and deployment to your own Firebase project + Cloud Run)
- `memory/PRD.md` — product requirements / feature notes
